3D Printable Patch Antenna

  • This is a research project focused on the design of 3D printable patch antennas for wearable 5G applications.
  • Polylactic Acid (PLA) and Polyurethane (PU) - the two flexible substrate substances used for the desired antenna design.
  • The patch antenna prototype was designed using CST Microwave Studio software.

A4 Sheet Vending Machine

  • This project implements Arduino IDE, and Embedded Programming using C++.
  • The project was designed using different modules like Arduino Uno, L298N Motor Drivers, and Infrared sensors.
  • The Arduino Uno served as the central control unit, while the L298N motor drivers facilitated precise movement and dispensing of items.

Motion Detection System

  • The basic structure of the project involves wireless interaction between a Client and a Server Module to transmit information.
  • The wireless communication is achieved with the help of NodeMCU programmed to transmit information to the server when the motion sensor is triggered.
  • This project implements different IoT techniques to improve security and efficiency of the system.
